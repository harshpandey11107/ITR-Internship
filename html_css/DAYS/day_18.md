## Python started

# 1. Introduction to Python

Python is a high-level, interpreted, and object-oriented programming language. It is simple, readable, and widely used for web development, AI, data science, automation, and software development.

# 2. Variables and Data Types
Variables store data in memory. Data Types:
int – Integer
float – Decimal numbers
str – Text
bool – True/False
list, tuple, set, dict

# 3. Operators
Used to perform operations on data.
Arithmetic (+, -, *, /)
Comparison (==, !=, >, <)
Logical (and, or, not)
Assignment (=, +=)
Membership (in, not in)

# 4. Conditional Statements
Used for decision-making.
if
elif
else

# 5. Loops
Used to repeat a block of code.
for loop
while loop

# 6. Functions
Functions are reusable blocks of code created using the def keyword.

# 7. Lists, Tuples, Dictionaries, and Sets
List: Ordered, mutable collection.
Tuple: Ordered, immutable collection.
Dictionary: Stores data as key-value pairs.
Set: Unordered collection of unique elements.

# 8. File Handling
Python can create, read, write, and append files using functions like open(), read(), write(), and close().

# 9. Exception Handling
Used to handle runtime errors using:
try
except
finally 

# 10. Modules and Packages
Module: A Python file containing functions and classes.
Package: A collection of related modules.

# 11. NumPy
NumPy (Numerical Python) is a Python library used for fast numerical calculations and working with arrays. It provides high-performance mathematical functions and supports multi-dimensional arrays.

# 12. Pandas
Pandas is a Python library used for data analysis and manipulation. It provides two main data structures:
Series – One-dimensional data.
DataFrame – Two-dimensional table (rows and columns).
