# Day 1

## Objective

To set up the development environment and understand the basics of Git and GitHub.

## Activities Performed

### 1. GitHub Account Setup

- Created and configured GitHub account.
- Understood the purpose of GitHub for code hosting and version control.

### 2. Git Installation

- Installed Git on the local system.
- Verified installation using Git commands.

### 3. VS Code Setup

- Installed Visual Studio Code.
- Configured the editor for web development.

### 4. Project Folder Structure Creation

* Created internship project folders:

  - HTML & CSS
  - JavaScript
  - React

### 5. GitHub Repository Creation

- Created GitHub repository for internship work.
- Connected local project with remote repository.

### 6. Basic Git Commands

* Learned and practiced:

  - git init
  - git add
  - git commit
  - git push
  - git status

### 7. Documentation Creation

- Created README.md file.
- Learned the importance of project documentation.

### 8. Introduction to Frontend Concepts

- Introduction to CSS Box Model.
- Introduction to CSS Flexbox.

## Learning Outcome

- Understood Git and GitHub workflow.
- Learned repository creation and code uploading process.
- Set up the development environment successfully.
- Gained basic knowledge of Box Model and Flexbox.
