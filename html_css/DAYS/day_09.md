# Day 9 - JavaScript Objects & Array Methods


 1. Dot and Bracket Notation

* Used to access object properties.
* Dot notation: `object.property`
* Bracket notation: `object["property"]`

 2. Computed Properties

* Allows dynamic property names using variables inside square brackets.
* Useful when property names are not fixed.

 3. Introduction to Objects

* Objects store data in **key-value pairs**.
* Help organize and manage related information efficiently.

 4. Iterating Objects

* `for...in` loop is used to iterate through object properties.
* Accesses each key and its corresponding value.

 5. Nested Destructuring

* Used to extract values from nested objects or arrays.
* Makes accessing deep data easier and cleaner.

 6. Object Destructuring

* Extracts object properties directly into variables.
* Reduces repetitive code and improves readability.

 7. Objects Inside Arrays

* Arrays can contain multiple objects.
* Commonly used for storing collections of related data.

 8. Spread Operator (`...`)

* Used to copy or expand arrays and objects.
* Helps create shallow copies and merge data.

 9. Array Methods

* Built-in functions for working with arrays.
* Used for adding, removing, searching, and modifying elements.

 10. `every()` Method

* Checks whether all elements satisfy a given condition.
* Returns `true` if every element passes the test; otherwise `false`.

 11. `fill()` Method

* Fills all or selected elements of an array with a specified value.

 12. `map()` Method

* Creates a new array by transforming each element of an existing array.

 13. `splice()` Method

* Adds, removes, or replaces elements in an array.
* Directly modifies the original array.

 14. `sort()` Method

* Sorts array elements.
* Can sort strings or numbers with a custom comparison function.
