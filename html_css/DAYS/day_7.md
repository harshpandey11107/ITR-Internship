
DAY 7 : JAVA SCRIPT

 1. Introduction to JavaScript:

JavaScript is a programming language used to make web pages interactive and dynamic. It helps add features like button clicks, form validation, animations, and content updates.

 2. JavaScript Fundamentals:

Learned the basic building blocks of JavaScript, including variables, data types, operators, conditions, and loops. These concepts form the foundation of web development.

 3. Variables:

Variables are used to store data values in JavaScript. They allow us to save and reuse information such as numbers, strings, and other data types.

Example:

```javascript
let name = "Harsh";
```

 4. Variable Keywords:

JavaScript provides three main keywords for declaring variables:

* `var` – Older way of declaring variables.
* `let` – Used when the value may change later.
* `const` – Used for values that should not be reassigned.

 5. String Indexing:

String indexing allows us to access individual characters of a string using their position. Indexing starts from `0`.

Example:

```javascript
let name = "harsh";
console.log(name[0]); // h
```

 6. String Methods:

String methods are built-in functions used to perform operations on strings, such as converting case, finding text, and extracting characters.

Example:

```javascript
let name = "harsh";
console.log(name.toUpperCase());
```

7. Truthy and Falsy Values:

In JavaScript, some values are automatically treated as `true` or `false` when evaluated in conditions.

Falsy values:

* false
* 0
* ""
* null
* undefined
* NaN

All other values are generally considered truthy.

8. Operators:

Operators are used to perform operations on values and variables.

* Arithmetic: `+`, `-`, `*`, `/`
* Comparison: `==`, `===`, `>`, `<`
* Logical: `&&`, `||`, `!`

9. Loops:

Loops are used to execute a block of code repeatedly until a specified condition is met.

Example:

```javascript
for(let i = 1; i <= 5; i++) {
    console.log(i);
}
```
