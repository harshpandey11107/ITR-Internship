


## What is CSS Grid?

CSS Grid is a two-dimensional layout system used to arrange elements in rows and columns. It helps create responsive and organized web page layouts with less code.

---

## Example

### HTML

```html
<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
</div>
